# YouTube Song Analyzer 🎵

Analyze songs from YouTube links! This Python project extracts the audio and detects key features like BPM, key, and more.

## Features
- Download audio from YouTube
- Detect BPM (beats per minute)
- Detect the key of the song
- Extract audio features

## Installation
1. Clone the repo: `git clone https://github.com/ATL07/youtube-song-analyzer.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Download [FFmpeg](https://ffmpeg.org/download.html) and set the path in `src/download.py`.

## Usage
Run: `python main.py` and enter a YouTube link.  
The script will show BPM, key, and more.
